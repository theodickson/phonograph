New Naming Conventions in {{ site.elusiveicons.minor_version }}

el-[name]-[shape]-[o]-[direction]

A desire for better icon naming consistency and predictability are at the heart of the changes in naming conventions. Once you know the name of the base icon shape, you should be able to predict the names of variants.

el-*

Solid icons as the base variant

  • el-bookmark
  • el-comment
  • el-folder

*-o

Outlined version of previous modifier

  • el-bookmark-o
  • el-comment-o
  • el-folder-o

*-circle

Circle under previous modifier

  • el-check-circle
  • el-exclamation-circle
  • el-plus-circle

*-square

Square under previous modifier

  • el-check-square
  • el-twitter-square
  • el-plus-square

*-right, *-up, etc.

Directional modifier always at the end

  • el-arrow-right
  • el-arrow-circle-right
  • el-angle-double-right

*-alt

Alternative to the original

  • el-unlock-alt
  • el-list-alt
  • el-github-alt

*-h, *-v

Horizontal & vertical modifiers

  • el-arrows-v
  • el-arrows-h
  • el-ellipsis-v

Combine 'em up...

Consistent when strung together

  • el-arrow-circle-o-right
  • el-caret-square-o-right
  • el-hand-o-right
